## Tiivistelmä SSH Public key


### SSH:n asennus ja käyttöönottto

Asennetaan OpenSSH <br></br>

*sudo apt-get update*

**sudo apt-get -y install ssh**
<br></br>

Käynnistetään ja otetaan palvelu käyttöön automaattisesti 
<br></br>
*sudo systemctl enable --now ssh*
<br></br>
